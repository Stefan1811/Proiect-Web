1. Introduction

This report will cover the development of the project "Baby Info Manager" . There will be discussed its purpose, what is does, how it interacts with hardware devices and what software technologies were used for implementation.

1.1 Purpose

Baby Info Manager is a project which aims at providing its users with information about taking care of a child in a proper way. Not only that but also having a tracking system that takes notice of a child sleep, medical visits and meals. The application is at its first iteration, the team hasn't developed anything similar before.

1.2 Product Scope

The application is going to track every aspect of a child's life, such as: sleep schedule, diet, medical history and social interactions.
The benefits of the application lay in having a controlled way of taking care of a child. Having such a vast amount of information about one's life makes things easier to get it on the right path by analyzing the knowledge given by the application. The main objective of Baby Info Manager is to make parenting easier, therefore giving a kid the best life they can have.

2. Overall description

2.1 Product Perspective

The application is a self-contained product. It is developed only by the team mentioned in this report. It is developed by using primarily HTML AND CSS for front-end, and PHP and MySQL for back-end.

2.2 Product Functions

The application will have the following functions:

  • Diet tracking and planning: the application will give its user the ability to plan meals in advance and also having a history of the meals served in the past.
  • Sleep tracking: the user will be able to monitor how many hours of sleep the child gets daily. The said hours will be introduced manually into the application.
  • Medical history: the user will be able to introduce conditions or diseases of the child if the case. It will also be possible to modify those aspects.
  • Social interactions: the user will be able to monitor the main people the child is interacting with.

2.3 Operating Environment

Baby Info Manager will be available on any device and operating system that has a functioning browser, whether it be a PC or portable device.

3. External Interface Requirements

3.1 User Interfaces

The application has the following pages which the user can interact with:

  • The Create Account page

    The user will be able to create an account via this page by introducing an email, a password, a first and a last name.

  • The Login page

    The user will be able to connect into their account by introducing the email and passoword they chose at registration.

  • The Parent Menu page

    After logging in, the user will be redirected to this page. Here lies the core functions of the application via which a child or more will be introduced in the system to be tracked.

  • The My Children page

    This page shows the names of the children that were introduced in the application by the parent. One of the names can be clicked in order to manage the information about the child.

  • The Child Menu page

    This page shows the all the information we can change,introduce and check about the child.

  • The Show Child Details page

    This page shws the child's name, sex, birthday, weight and height.

  • The Medical History page

    This page allows the user to check the child's medical visits summaries introduced by the parent.

  • The Medical Visit Add page

    This page allows the user to add to the child's medical visits summaries.

  • The Friends page

    This page allows the user check the child's friendships, acquaintances and cousins.

  • The Friends Add page

    This page allows the user to add to the child's friendships, acquaintances and cousins.

  • The Sleep History page

    This page allows the user to check the child's slept hours whenever it has been introduced by the parent.

  • The Add Sleep page

    This page allows the user to check the add to the child's slept hours.

  • The Meal History page

    This page allows the user to check the add to the child's eaten meals whenever they have been introduced by the parent.

  • The Add Meal page

    This page allows the user to add to the child's eaten meals.

  • The Change Child's info page

    This page allows the user to change the information initially introduced by the parent.

  • The Pictures page

    This page allows the user to upload pictures in the application and to whether set them as favorite or delete them. By clikcing the "Favorites" button there will be shown only the pictures selected as favorites, and by clicking "Not Favorites" there will be shown all the updated pictures in the app of that child's account.

  • The Add Child page

    This page allows the user to add the child to the "My Children" page after their accounts have been created.

  • The Create Child Account page

    This page allows the user to create an account for one of its intended childern to be tracked by the application.

  • The Change Password page

    This page allows the user to the password of their account.

4. System Features

4.1 Friends, Sleep, Meal, Doctor visits track

Baby Info Manager application was built in mind with this section's listed features. For instance, if a child starts to be sickly all of a sudden a parent might not know exactly where that comes from. By tracking the people the child interacts with, what they eat and when, when they sleep and for how much and what the doctor says at each visit, for sure it will be easier for the parent to spot a dead angle when it comes to their child's unrest. The application was made to give as much freedom to the parent as there is to give. They are not constrained by any predefined meals, hours of sleep or doctor visits.

4.2 Saving important pictures of the child

Baby Info Manager application provides the parent with the possibility to save important pictures of their child right into the application itself. Those pictures are private and can only be seen by the one with the account.

5. Conclusions

To summarize, Baby Info Manager application aims at providing its users with the best version of the features described that help parents to keep track of every aspect of their children's lives.
